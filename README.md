# Emotion & Polarity Analysis of Social Media Comments

## Project Description
This project analyzes user comments and identifies:
- Emotion (Happy, Sad, Angry, etc.)
- Polarity (Positive, Negative, Neutral)

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- NLP: TextBlob

## Features
- Real-time comment analysis
- Emotion detection
- Polarity classification
- Simple UI interface
